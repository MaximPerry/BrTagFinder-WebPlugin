Features

SlashOn brTag Finder has been made to easily reveal the all <br /> tags in a web page.

  • Reveals the <br /> tags by inserting a small image right before;
  • Does not change the CSS style of the web page;
  • Outputs the total count of <br /> tags in the web console.
